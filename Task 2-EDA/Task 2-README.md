# Task 2: Exploratory Data Analysis (EDA)

## 🎯 Objective
To clean the raw scraped data from Zameen.pk and identify key market trends in Lahore's real estate sector.

## 🧹 Data Cleaning Process
- **DHA Grouping:** Merged multiple DHA phases into a unified 'DHA, Lahore' category.
- **Price Conversion:** Transformed 'Crore' strings into numeric values for calculation.
- **Outlier Handling:** Filtered out 'Consult Agent' entries to ensure accurate averages.

## 📈 Statistical Results
- **Total Dataset:** 174 properties.
- **Average Price:** PKR 261.6 Million.
- **Median Price:** PKR 250.0 Million (This indicates a balanced luxury market).
- **Highest Price:** PKR 500.0 Million.

## 📍 Top Market Locations
1. **DHA, Lahore:** 56 listings (Dominant Market Leader)
2. **Samanabad:** 10 listings
3. **Allama Iqbal Town:** 6 listings
