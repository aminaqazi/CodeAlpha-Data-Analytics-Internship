# Lahore Real Estate Market Analysis: Zameen.pk
**Internship:** CodeAlpha Data Analytics  
**Author:** Amina Qazi

## 📌 Project Overview
This project is a end-to-end data science pipeline designed to analyze house prices in Lahore. It covers everything from raw data collection via web scraping to deep statistical analysis and visualization.

## 📁 Project Structure
The project is organized into three distinct phases:

- **[Task 1: Web Scraping](./Task%201-Web%20Scraping/):** Automating data collection using SeleniumBase.
- **[Task 2: EDA](./Task%202-EDA/):** Cleaning "messy" real estate data and calculating market statistics.
- **[Task 3: Visualization](./Task%203-Visualization/):** Creating charts to identify market trends.
